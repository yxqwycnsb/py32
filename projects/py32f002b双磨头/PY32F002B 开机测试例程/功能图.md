## 2. 电机控制状态机

```mermaid
stateDiagram-v2
  [*] --> SWITCH_OFF

  SWITCH_OFF: 船型开关断开
  SWITCH_OFF: PA0=0 PB3=0
  SWITCH_OFF: 清除 fault

  SWITCH_OFF --> RUNNING: 开关接通(PB7=0) 且 fault=0

  RUNNING: 电机运行
  RUNNING: PA0=1 PB3=1
  RUNNING --> LATCHED_OFF: 连续3次过流(I>=5A)
  RUNNING --> SWITCH_OFF: 开关断开(PB7=1)

  LATCHED_OFF: 锁存关机
  LATCHED_OFF: PA0=0 PB3=0
  LATCHED_OFF --> SWITCH_OFF: 必须关开关清锁存
```

## 4. 关键保护点

- **过流保护**：
  - 阈值：`I >= 5A`
  - 连续确认：`3` 次（20ms 周期）
  - 动作：立即关闭电机并锁存
  - 恢复：必须船型开关 OFF 清除锁存

- **看门狗 IWDG**：
  - 超时：约 2s
  - 喂狗：200ms
  - 目的：防止软件死锁导致电机失控
