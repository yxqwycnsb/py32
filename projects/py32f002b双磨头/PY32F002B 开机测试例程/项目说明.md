# PY32F002B 电机控制与电池管理系统说明

## 1. 项目概述
本项目基于 **PY32F002B** 芯片，采用 **Protothread** 协作式多任务框架，实现了一个集成了电机智能控制、堵转保护、电池电量监测及充电状态指示的控制系统。

## 2. 硬件引脚分配 (Pinout)

| 模块 | 信号名称 | MCU 引脚 | 模式/有效电平 |
| :--- | :--- | :--- | :--- |
| **电机控制** | Motor_EN | **PA0** | 输出 / 高电平开启电机 |
| **指示灯** | Motor_LED (G) | **PB3** | 输出 / 高电平亮 (电机运行时亮) |
| **指示灯** | Charge_LED (R) | **PA7/PB4** | 输出 / 充电指示、低压报警 |
| **控制开关** | Rocker_Switch | **PB7** | 输入 / 上拉，低电平表示闭合(ON) |
| **充电芯片** | TP4056_CHRG | **PB2** | 输入 / 上拉，低电平表示正在充电 |
| **电流采样** | I_SENSE | **PA3** | ADC 通道 1 / 接 5mΩ 采样电阻 |
| **电压采样** | VREFINT | **内部** | 内部 1.5V 参考源 (用于反算 VDD) |
| **调试串口** | UART TX/RX | **PB4/PB5** | 115200 8N1 (PB4 与红灯冲突时优先 UART) |

## 3. 核心功能逻辑

### 3.1 电机控制与复位 (Strict Reset)
*   **启动**：船型开关物理闭合 (ON) $\rightarrow$ 电机启动。
*   **堵转停机**：检测到电流 **> 6000mA** 且持续 **2 秒**，电机立即停机并锁定故障。
*   **物理复位**：堵转后即便开关还闭合，电机也不转。必须**手动断开开关 (OFF)** 才能清除故障锁存；再次闭合后电机重新启动。

### 3.2 电池电压管理 (VREFINT 反算法)
*   **测量原理**：由于 VDD 直连电池，程序通过测量内部稳定的 **1.5V 参考电压** 反推当前电池电压，解决 Vref 随电池波动的难题。
*   **低压保护**：电压 **< 3.3V** 时强制关停电机。
*   **尝试启动报警**：低压时若拨动开关至 ON，红灯将**闪烁 5 次** (0.5s 间隔) 提醒，电机不启动。

### 3.3 红灯指示状态优先级
1.  **最高优先级**：低压报警 (3.3V 以下尝试启动) $\rightarrow$ **红灯闪烁 5 次**。
2.  **次高优先级**：充电中且电池 **< 4.1V** $\rightarrow$ **红灯呼吸灯效果**。
3.  **普通优先级**：充电中且电池 **≥ 4.1V** $\rightarrow$ **红灯常亮**。
4.  **无充电**：红灯熄灭。

## 4. 串口调试指令 (UART Commands)
串口参数：`115200 8N1`。所有命令需带回车。

| 命令 | 功能说明 |
| :--- | :--- |
| **`out m`** | 远程开启电机（需开关在 ON 位且无故障） |
| **`off m`** | 远程关闭电机 / 手动清除堵转故障 |
| **`adc m zero`** | **电流校零**：电机停止时执行，记录背景噪声 |
| **`adc v on/off`** | 开启或关闭电池电压的实时打印 |

### 状态报告格式：
电机运行时，每 1 秒自动打印一次：
`STATUS: Vbat=4150mV(RawV:1480) Motor=800mA(RawM:1520, Offs:1500)`

---

## 5. 安全保护参数
*   **堵转阈值**：6000mA (6A)。
*   **堵转确认时间**：2000ms (2s)。
*   **低压关断阈值**：3300mV (3.3V)。
*   **充满判定阈值**：4100mV (4.1V)。
*   **看门狗 (IWDG)**：约 2 秒超时，自动喂狗。
